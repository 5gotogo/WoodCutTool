FAQ

Is iCloud sync a backup?

Not really. Sync mirrors the current state; a problem can propagate. An exported file is a true archive.

Can I move contacts to a non-Apple phone?

Yes. CSV and vCard exports are cross-platform, unlike iCloud-only contacts.

Is the export tied to my Apple account?

No. The file is independent of iCloud and stored by you.

Do I still need iCloud if I export?

iCloud is fine for daily sync; the export adds portability and a recoverable archive.
