How to optimize a plywood cut list online

Use this browser calculator for quick cabinet, shelf, drawer, shop fixture, and plywood project estimates. The same workflow maps directly to CutList for iPhone when you need saved projects, PDF export, AirPrint, or offline job planning.

  1. Set stock sheet size and saw kerf. Enter the board or plywood sheet length, width, thickness, and the blade kerf that should be removed between cuts.
  2. Enter each required part. Add the part name, finished length, finished width, and quantity so repeated panels are counted correctly.
  3. Add repeated shelves, sides, rails, and panels. Use Add part for every component that must come from the same stock material.
  4. Run the cut list optimizer. Select Optimize cut list to calculate a board-by-board cutting sequence in the browser.
  5. Review boards required and waste. Check the boards required, cut order, and waste percentage, then move saved or printable projects into the CutList iPhone app.
