# SEO Titles & Meta Descriptions — woodcuttool.com

Goal: improve Google CTR. For each page below: **3 title variations**, a recommended
pick, and **1 meta description under 155 characters** with the target keyword used
naturally.

**How to use these**
- Titles aim for **≤ 60 chars** so Google doesn't truncate them (current site has several
  84–91 char titles that get cut off in results).
- Descriptions are **≤ 155 chars** (Google truncates ~155–160 on desktop). Current
  homepage description is 195 chars — it is being truncated today.
- The 3 variations follow distinct angles so you can A/B test:
  **(A) keyword-first / literal**, **(B) benefit-led**, **(C) modifier or question** (Free,
  2026, numbers, "How…?") — modifiers and numbers reliably lift CTR.
- `| WoodCutTool` brand suffix is optional; drop it on long titles to save space.
- Char counts shown as `[n]`.

Legend: ★ = recommended variation.
