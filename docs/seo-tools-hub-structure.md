# WoodCutTool SEO Hub Structure

Primary hub URL: `/tools/`

Primary keywords:
- woodworking calculator
- cut list optimizer
- material calculator
- DIY construction tools

Internal linking rule:
- `/tools/` links to every tool and app page.
- Every tool and app page links back to `/tools/` through top navigation or footer navigation.
- High-priority woodworking pages should also link laterally to related calculators, guides, and CutList app pages.

## Hub Page

URL: `/tools/`

H1: Woodworking Calculator, Material Calculator, and DIY Construction Tools

SEO description: WoodCutTool's tools hub is the main topic cluster page for woodworking calculators, material calculators, DIY construction tools, and privacy-first productivity apps. It connects high-intent users to the CutList cut list optimizer, plywood layout planning tools, wood waste calculators, board foot calculators, stair stringer calculators, tile calculators, document scanners, business utilities, and personal productivity apps. The page should target users who are comparing practical tools before starting a project, buying materials, cutting plywood, estimating construction quantities, scanning documents, tracking business records, or organizing everyday work. The hub should rank for broad discovery terms and pass internal authority to specialized pages through clear category sections, descriptive anchor text, and direct links to each tool.

Linking strategy:
- Hub links down to all category tools.
- Tool pages link back up with anchor text such as "All WoodCutTool tools".
- Related woodworking pages cross-link between CutList, plywood calculator, cut list calculator, waste calculator, board foot calculator, and plywood guide.
