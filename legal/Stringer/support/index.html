Stringer Support

Support page for Stringer, an iPhone utility app that designs stairs and stair stringers with code checks and a printable cut sheet, fully offline.

App Overview

Stringer is a utility and productivity app for carpenters, deck builders, remodelers, and DIY users who need to lay out a stair stringer and check it against a building code directly on iPhone.

CategoryUtilities / Productivity
PlatformiOS (iPhone)
Version1.0.0
PrivacyOffline project workflow

Main Features

  • Enter total rise and going (tread run) to calculate balanced riser and tread options.
  • Choose a building code (IRC US/CA, NCC AU, or Doc K UK) and see compliance checks for riser height, tread depth, pitch, and stair width.
  • Compare all valid riser counts with run, pitch, and a recommended best option.
  • View the stair elevation and the stringer layout, including plumb and level cut angles, first-cut, throat depth, and board length.
  • Set construction options: standard or flush top, layout, nosing, tread thickness, dropped stringer, stringer stock, and stair width.
  • Estimate stringer stock cost from a price per length.
  • Export, print, or share a stair cut sheet only when you choose an iOS system action.

Core User Flow

Step 1Enter total rise and tread run, then pick a building code.
Step 2Calculate, then compare the riser and tread options and choose one.
Step 3Review the stringer layout and code checks, then optionally export or print the cut sheet.

Building Code Notes

Code references such as IRC R311.7, NCC, and Doc K are provided for general estimating and planning support only. Compliance results, including the headroom check, are estimates and must be verified on site. Always confirm measurements and local code and permit requirements with a qualified professional before building.

Privacy and Data Handling

Project data is processed offline on the user's device. Stringer does not use a remote calculation server, public profiles, social feeds, or user-generated community content. See the Stringer Privacy Policy.
